### 1.2 使用模型的代码

**创建测试脚本**：

```bash
cd /home/mfj/robosuite/my_experiments/baseline_lift_ppo
cat > test_model.py << 'EOF'
"""
===============================================================================
                    测试训练好的 PPO 模型
===============================================================================
"""

from stable_baselines3 import PPO
import robosuite as suite
import numpy as np
import time

print("=" * 70)
print("测试训练好的 PPO 模型")
print("=" * 70)

# =============================================================================
# 第 1 步：加载模型
# =============================================================================

model_path = "ppo_lift_final_v2.zip"  # 可以改成 v3
print(f"加载模型：{model_path}")
model = PPO.load(model_path)
print("✅ 模型加载成功！")

# =============================================================================
# 第 2 步：创建环境（带渲染）
# =============================================================================

print("\n创建环境（带图形界面）...")

env = suite.make(
    "Lift",
    robots="Panda",
    has_renderer=True,           # 显示图形界面
    has_offscreen_renderer=True,
    use_camera_obs=False,
    use_object_obs=True,
    reward_shaping=True,
    control_freq=20,
    horizon=500,
)

print("✅ 环境创建成功！")

# =============================================================================
# 第 3 步：运行测试
# =============================================================================

print("\n开始测试（按 Ctrl+C 停止）...")
print("观察机器人是否能成功举起方块！")

total_reward = 0
success_count = 0

for episode in range(10):  # 测试 10 次
    obs = env.reset()
    episode_reward = 0
    
    print(f"\n--- 第 {episode+1} 次尝试 ---")
    
    for step in range(500):
        # 使用模型预测动作
        action, _ = model.predict(obs, deterministic=True)
        
        # 执行动作
        obs, reward, done, info = env.step(action)
        episode_reward += reward
        
        # 渲染显示
        env.render()
        time.sleep(0.02)  # 放慢速度方便观察
        
        if done:
            print(f"✅ 成功！步数：{step}, 奖励：{episode_reward:.2f}")
            success_count += 1
            break
    
    total_reward += episode_reward

# =============================================================================
# 第 4 步：打印统计
# =============================================================================

print("\n" + "=" * 70)
print("测试完成！统计结果：")
print("=" * 70)
print(f"测试次数：10 次")
print(f"成功次数：{success_count} 次")
print(f"成功率：{success_count/10*100:.1f}%")
print(f"平均奖励：{total_reward/10:.2f}")

env.close()
print("\n测试结束！")
EOF

echo "✅ 测试脚本创建成功！"
```

### 1.3 运行测试


## 🎯 问题 3：作用的对象是谁？

### 3.1 当前环境

**当前训练的环境**：
- **任务**：Lift（举起方块）
- **机器人**：Panda（Franka Emika Panda 机械臂）
- **对象**：一个立方体方块
- **目标**：把方块从桌面举起

### 3.2 观测值（输入）

**模型看到的"世界"**（84 维向量）：

| 类别 | 内容 | 维度 |
|------|------|------|
| 机器人状态 | 关节角度、角速度 | ~16 维 |
| 夹爪状态 | 开合程度、速度 | ~4 维 |
| 方块状态 | 位置、速度、旋转 | ~6 维 |
| 相对位置 | 手到方块的距离 | ~3 维 |
| 其他 | 接触力、奖励信号等 | ~55 维 |

### 3.3 动作（输出）

**模型输出的动作**（8 维向量）：

| 维度 | 控制内容 | 范围 |
|------|----------|------|
| 0-6 | 7 个关节的力矩 | [-1, 1] |
| 7 | 夹爪开合 | [-1, 1] |

### 3.4 总结

```
输入（84 维观测） → 神经网络 → 输出（8 维动作）
                          ↓
                    机器人执行动作
                          ↓
                    方块被举起
```

## 🦾 问题 4：能部署到真正的机械臂上吗？

### 4.1 答案：可以，但需要额外工作！

**训练好的模型可以部署到真实机械臂**，但需要：

1. **相同的机器人型号**（Panda）
2. **真实传感器数据**（代替仿真观测）
3. **安全机制**（防止碰撞）
4. **校准**（仿真与现实的差距）

### 4.2 部署流程

```
┌─────────────────────────────────────────────────────────────┐
│ 第 1 步：准备真实机器人                                      │
│ - 安装 Panda 机械臂                                          │
│ - 连接控制电脑                                               │
│ - 配置网络通信                                               │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 第 2 步：创建真实环境接口                                     │
│ - 读取真实传感器数据（关节角度、力矩等）                     │
│ - 发送控制命令到机械臂                                       │
│ - 处理安全限制                                               │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 第 3 步：加载训练好的模型                                     │
│ model = PPO.load("ppo_lift_final_v2.zip")                   │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 第 4 步：运行控制循环                                         │
│ while True:                                                  │
│     obs = 读取真实传感器 ()                                   │
│     action = model.predict(obs)                              │
│     发送动作到机械臂 ()                                       │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 需要的硬件

| 硬件 | 说明 | 预估价格 |
|------|------|----------|
| Franka Panda 机械臂 | 7 自由度协作机器人 | ~1 万美元 |
| 夹爪 | 平行夹爪 | 包含在机械臂中 |
| 控制电脑 | 运行模型的电脑 | 已有 |
| 摄像头（可选） | 视觉观测 | ~100 美元 |
| 方块 | 任务对象 | 自制 |

### 4.4 仿真与现实的差距

**问题**：仿真中训练的模型在现实中可能表现不好

**原因**：
- 仿真物理不完美
- 传感器噪声
- 机械臂校准误差
- 摩擦模型不准确

**解决方案**：
1. **域随机化**：在仿真中随机化参数（摩擦、质量等）
2. **微调**：在真实机器人上继续训练
3. **系统辨识**：校准仿真参数匹配真实机器人

## ⚙️ 问题 5：需要改什么参数？

### 5.1 从仿真到真实机器人的修改

**创建真实机器人接口**：

```python
# 仿真环境（当前使用）
class RobosuiteGymWrapper(gym.Env):
    def __init__(self):
        self.env = suite.make("Lift", robots="Panda", ...)
    
    def reset(self):
        obs = self.env.reset()  # 仿真返回观测
        return obs_vec, {}
    
    def step(self, action):
        obs, reward, done, info = self.env.step(action)  # 仿真执行
        return obs_vec, reward, done, False, info

# 真实机器人环境（需要创建）
class RealPandaEnv(gym.Env):
    def __init__(self):
        # 连接真实机器人
        self.robot = connect_to_panda(ip="192.168.1.100")
        self.camera = connect_to_camera()
    
    def reset(self):
        # 重置机器人到初始位置
        self.robot.move_to_home()
        # 放置方块到初始位置（需要外部机构或人工）
        place_cube()
        # 读取初始观测
        obs = self.read_sensors()
        return obs, {}
    
    def step(self, action):
        # 发送动作到机器人
        self.robot.send_torque_command(action)
        # 等待执行
        time.sleep(0.05)
        # 读取新观测
        obs = self.read_sensors()
        # 计算奖励（需要定义）
        reward = self.compute_reward()
        # 检查是否成功
        done = self.check_success()
        return obs, reward, done, False, {}
    
    def read_sensors(self):
        # 读取关节角度
        joint_angles = self.robot.get_joint_angles()
        # 读取夹爪状态
        gripper_pos = self.robot.get_gripper_pos()
        # 读取方块位置（从摄像头）
        cube_pos = self.camera.detect_cube()
        # 拼成向量
        obs_vec = np.concatenate([joint_angles, gripper_pos, cube_pos, ...])
        return obs_vec
```

### 5.2 需要修改的参数

| 参数 | 仿真值 | 真实机器人值 | 说明 |
|------|--------|-------------|------|
| `control_freq` | 20 Hz | 20 Hz | 控制频率 |
| `has_renderer` | True/False | False | 真实不需要渲染 |
| `use_camera_obs` | False | True（可选） | 真实可以用摄像头 |
| 观测维度 | 84 | 84（需匹配） | 必须相同 |
| 动作维度 | 8 | 8 | 必须相同 |
| `horizon` | 500 | 500 | 最大步数 |

### 5.3 安全参数（真实机器人特有）

```python
# 安全限制
MAX_TORQUE = 50.0  # 最大力矩（Nm）
MAX_SPEED = 0.5    # 最大速度（m/s）
EMERGENCY_STOP = True  # 急停开关

# 碰撞检测
COLLISION_THRESHOLD = 10.0  # 碰撞检测阈值

# 工作空间限制
WORKSPACE_BOUNDS = {
    'x': [-0.5, 0.5],
    'y': [-0.5, 0.5],
    'z': [0.0, 0.8],
}
```

### 5.4 完整部署代码框架

```python
from stable_baselines3 import PPO
import numpy as np

# 加载模型
model = PPO.load("ppo_lift_final_v2.zip")

# 连接真实机器人
robot = RealPandaEnv()

# 运行控制循环
obs = robot.reset()
for step in range(500):
    # 模型预测
    action, _ = model.predict(obs, deterministic=True)
    
    # 安全限制
    action = np.clip(action, -1, 1)
    
    # 执行动作
    obs, reward, done, _ = robot.step(action)
    
    # 检查成功
    if done:
        print("任务成功！")
        break

robot.close()
```
