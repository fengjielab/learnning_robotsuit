### 2️⃣ `ppo_logs/monitor.csv` - 训练监控日志

**作用**：记录每一集的训练数据

**查看方式**：
```bash
# 直接查看
cat ppo_logs/monitor.csv

# 或用 Excel 打开
libreoffice ppo_logs/monitor.csv
```

**内容格式**：
```csv
#{"t_start": 1234567890.123, "env_id": "..."}
r,l,t
15.234,500,0.123
18.567,500,1.456
21.890,500,2.789
...
```

**列说明**：
- `r` (reward)：这一集的总奖励
- `l` (length)：这一集的步数
- `t` (time)：累计时间（秒）

---

### 3️⃣ `ppo_tensorboard/` - TensorBoard 数据目录

**作用**：存储 TensorBoard 可视化数据

**目录结构**：
```
ppo_tensorboard/
└── lift_ppo_run1_1/
    └── events.out.tfevents.1234567890.hostname.pid
```

**查看方式**：
```bash
cd /home/mfj/robosuite/my_experiments/baseline_lift_ppo
# 启动 TensorBoard
tensorboard --logdir ./ppo_tensorboard

# 浏览器打开
http://localhost:6006
```

**可以看到的指标**：

| 指标名称 | 含义 | 期望趋势 |
|----------|------|----------|
| `rollout/ep_rew_mean` | 平均奖励 | 上升 ↗️ |
| `rollout/ep_len_mean` | 平均回合长度 | 稳定 |
| `train/loss` | 总损失 | 下降 ↘️ |
| `train/value_loss` | 价值网络损失 | 下降 ↘️ |
| `train/policy_gradient_loss` | 策略损失 | 波动 |
| `train/entropy` | 策略熵（探索程度） | 逐渐下降 |
| `train/learning_rate` | 学习率 | 恒定 |
| `train/approx_kl` | KL 散度近似 | 小值波动 |
| `train/clip_fraction` | 被裁剪的比例 | 接近 0 |

---

### 4️⃣ `ppo_checkpoints/` - 训练检查点

**作用**：定期保存模型，可以从中断点恢复训练

**目录结构**：
```
ppo_checkpoints/
├── ppo_lift_10000_steps.zip    # 1 万步的模型
├── ppo_lift_20000_steps.zip    # 2 万步的模型
├── ppo_lift_30000_steps.zip    # 3 万步的模型
├── ...
└── ppo_lift_100000_steps.zip   # 10 万步的模型
```

**使用方式**：
```python
from stable_baselines3 import PPO

# 加载检查点继续训练
model = PPO.load("./ppo_checkpoints/ppo_lift_50000_steps")
model.learn(total_timesteps=150000)  # 继续训练到 15 万步
```

---

### 5️⃣ `ppo_lift_final.zip` - 最终模型

**作用**：训练完成的最终模型，可以直接使用

**使用方式**：
```python
from stable_baselines3 import PPO

# 加载模型
model = PPO.load("ppo_lift_final")

# 使用模型预测动作
obs = env.reset()
action, _ = model.predict(obs, deterministic=True)
```

## 🔍 常见问题

### Q1: 训练中断了怎么办？

**A**: 从检查点恢复训练：
```python
from stable_baselines3 import PPO

# 加载最近的检查点
model = PPO.load("./ppo_checkpoints/ppo_lift_50000_steps")

# 继续训练
model.learn(total_timesteps=100000)  # 从 5 万步继续到 10 万步
```

---

### Q2: 如何比较多次训练的结果？

**A**: 在 TensorBoard 中比较：
```bash
cd /home/mfj/robosuite/my_experiments/baseline_lift_ppo
# 启动 TensorBoard，指定多个运行目录
tensorboard --logdir ./ppo_tensorboard

# 在浏览器中可以看到多条曲线，每条代表一次运行
```

---

### Q3: 训练效果不好怎么办？

**A**: 查看 TensorBoard 分析：
- 如果奖励不上升 → 增加训练步数或调整学习率
- 如果损失不下降 → 检查网络结构或奖励函数
- 如果波动太大 → 减小学习率或增加 batch_size

---

### Q4: 如何保存训练日志供以后分析？

**A**: 备份整个目录：
```bash
# 复制所有训练数据
cp -r ppo_logs ppo_tensorboard ppo_checkpoints ppo_lift_final.zip \
   ~/backup/training_run1/
```
